## 데모 시나리오 테스트

### 시나리오 1: 첫 만남 + 테라리움 감상 (1분)

**단계**:
1. 앱 실행
2. 3D 테라리움 로드 확인
3. 초록이 인사 확인 ("안녕, 나는 초록이야 🌱")
4. 정령 상태 "고요" 표시 확인
5. 테라리움 터치 회전
6. 10초 후 코인 +1 확인

**체크리스트**:
- [ ] 3D 씬이 4초 내 로드됨
- [ ] 유리병 + 흙 표시됨
- [ ] 자동 회전 동작
- [ ] 터치로 회전 조작 가능
- [ ] 코인 카운터 표시됨

---

### 시나리오 2: 정령과 대화 (1분 30초)

**단계**:
1. 💬 대화 버튼 탭
2. 채팅 바텀시트 열림
3. 오늘의 질문 표시 ("오늘 하루 어땠어?")
4. "힘들었어" 입력
5. 정령 응답 확인 (스트리밍)
6. 추가 대화 1턴

**체크리스트**:
- [ ] 바텀시트 부드럽게 열림
- [ ] 오늘의 질문 랜덤 표시
- [ ] 응답이 타이핑되듯 나타남 (스트리밍)
- [ ] **응답에 조언 없음** ⚠️
- [ ] **"매일 해봐" 같은 부담 주는 말 없음** ⚠️
- [ ] 응답 50자 이내

---

### 시나리오 3: 꾸미기 + 마무리 (1분)

**단계**:
1. 🛒 상점 버튼 탭
2. 상점 바텀시트 열림
3. 버섯 아이템 확인 (감정 기능: "기운이 필요할 때")
4. 버섯 구매 (12코인)
5. 초록이 반응 확인 ("오늘 힘이 필요했구나 🍄")
6. 테라리움에 버섯 3D 표시 확인
7. "오늘은 여기까지" 버튼 탭
8. 마무리 멘트 표시
9. 1줄 일기 후보 2개 표시
10. 선택 후 저장

**체크리스트**:
- [ ] 아이템 카드에 감정 기능 표시됨
- [ ] 구매 시 코인 차감됨
- [ ] 정령 반응 멘트 표시됨
- [ ] **버섯 3D가 테라리움에 추가됨**
- [ ] 마무리 멘트 표시됨
- [ ] 일기 후보 2개 중 선택 가능
- [ ] AsyncStorage에 저장됨
